# LocalTeam

Open-source desktop application for orchestrating AI agent teams.

## Development

### Prerequisites
- [Rust](https://rustup.rs/) stable (1.77.2+)
- [Node.js](https://nodejs.org/) (20+)
- Windows 10/11

### Setup
```powershell
npm install
npm run tauri dev
```

The default project configuration lives in `localteam.json`. Runtime session state is written to `.localteam/`.
